# sodanhamaOS

## Description

This project serves as my personal, web-based operating system. It features the ability to see the time, and to listen to a grand total of 2 of my favourite Minecraft OSTS. Windows, once opened, are draggable, and can be closed.
I was inspired by liquid glass, and other much more advanced websites.

## How I made this project
I made this project using HTML, CSS, and JS. I took inspiration from Hack Club's WebOS Jam. I used backgrounds from Minecraft's official website, and music from Lena Raine, produced for Minecraft.
